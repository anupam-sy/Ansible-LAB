Apache2Installation_Playbook
============================
A playbook to install Apache2 web server on Ubuntu systems.

HOW TO RUN PLAYBOOK:
====================
>>> To DRY Run the playbook (Means, To make sure everything will 
execute as expected), Use the following command:

    ansible-playbook apache2.yml --check

>>> To actually run the playbook, use the following command:

    ansible-playbook apache2.yml
