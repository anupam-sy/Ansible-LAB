ExceptionHandling_Playbook
==========================
A playbook to implement exception handling and executing different
task on different servers of hosts group.

HOW TO RUN PLAYBOOK:
====================
>>> To DRY Run the playbook (Means, To make sure everything will 
execute as expected), Use the following command:

    ansible-playbook <playbook_name> --check

>>> To actually run the playbook, use the following command:

    ansible-playbook <playbook_name>
