SystemSetUP_Playbook
=====================
A playbook to install some essentials to smoothly use linux system.

HOW TO RUN PLAYBOOK:
====================
>>> To DRY Run the playbook (Means, To make sure everything will 
execute as expected), Use the following command:

    ansible-playbook systemsetup.yml --check

>>> To actually run the playbook, use the following command:

    ansible-playbook systemsetup.yml
