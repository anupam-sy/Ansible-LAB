UserCreation_Playbook
=====================
A playbook to create a normal user on Ubuntu systems.

HOW TO RUN PLAYBOOK:
====================
>>> To DRY Run the playbook (Means, To make sure everything will 
execute as expected), Use the following command:

    ansible-playbook usercreation.yml --check

>>> To actually run the playbook, use the following command:

    ansible-playbook usercreation.yml
